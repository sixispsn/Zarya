# Нормативная матрица пояснительной записки ИОС2 (стадия П)

Актуальный для проекта набор источников:

- постановление Правительства РФ № 87, раздел 5, пункт 17 - обязательный состав подраздела «Система водоснабжения»;
- ГОСТ Р 21.619-2023 - состав и правила представления рабочей документации внутренних систем водоснабжения и водоотведения;
- СП 30.13330.2020 - расчетные методы, ограничения и конструктивные требования внутренних систем водоснабжения и водоотведения.

Принцип трассируемости: каждое число в ПЗ должно иметь путь `исходные данные -> формула/алгоритм -> результат -> текст/таблица/схема -> тест`. Отсутствующие исходные данные показываются как незавершенность расчета; расчетные значения по умолчанию не подставляются.

## Расчетная цепочка Hтр и насосной установки

1. Расходы В1/Т3 рассчитываются из групп потребителей.
2. Водомер выбирается по эксплуатационному расходу и проверяется по потерям, пожарному режиму и чувствительности (СП 30, 12.14-12.17).
3. Потери выбранного диктующего узла учета входят в Hтр.
4. `Hтр = Hgeom + ΣHil + Hпр + ΣHвод + Hтепл + Hlввод` (СП 30, 8.27, формула (14)).
5. Необходимость насоса определяется только сравнением Hтр с гарантированным напором из ТУ.
6. Рабочая точка насоса определяется по кривой системы `Hнас(Q) = Hстат + kQ²`, проходящей через расчетную точку `Qрасч; Hтр - Hгар`.
7. В ПЗ выдаются тип, рабочая точка Q/H, мощность, резервирование, NPSHr и результат проверки NPSHa. Если NPSHa не задан, вывод маркируется как требующий проверки, а не считается положительным.

## Ближайшие незакрытые блоки

1. Зонный расчет с проверкой минимального и максимального давления на приборах каждой зоны.
2. Автоматический подбор диаметров В1 и поддержка кольцевых/многовводных хозяйственно-питьевых сетей; текущая топология В1 является ориентированным деревом с заданными диаметрами.
3. Теплогидравлический расчет Т3/Т4 и потерь в теплообменнике по паспортным данным.
4. Полный баланс по группам потребителей и форма 2.
5. Раздельные принципиальные схемы здания, насосной станции и водомерного узла по 5.2.2.3-5.2.2.6 ГОСТ Р 21.619-2023.
