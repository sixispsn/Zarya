# Zarya Demo Release 0.1

Папка формируется командой:

```bash
OSIFONT_PATH=/System/Library/Fonts/Supplemental/Arial.ttf venv/bin/python scripts/build_demo_release.py
```

Состав: исходный YAML, пять PDF и `manifest.json` с контрольными суммами и ключевыми результатами расчёта.

Все исходные реквизиты с префиксом `DEMO` являются демонстрационными и не относятся к реальному объекту строительства.
