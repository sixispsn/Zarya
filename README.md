# Заря

Инженерный калькулятор внутреннего водоснабжения и канализации по СП 30.13330.2020. Рассчитывает водопотребление, В1, В2, Т3/Т4, водоотведение, водостоки, водомеры, теплоизоляцию и насосные установки; формирует комплект документов стадии П.

## Запуск для проверки

Требуются Python 3.12 и доступ в интернет при первом запуске. Само приложение работает локально: данные не отправляются на внешний сервер.

- macOS: распаковать архив и открыть `Запустить Зарю.command`. Если macOS заблокирует загруженный файл, открыть его через правый клик → «Открыть».
- Windows: распаковать архив и открыть `Запустить Зарю.bat`.
- После подготовки окружения браузер откроет `http://127.0.0.1:8000/wizard` автоматически.
- Для остановки закрыть окно запуска или нажать `Ctrl+C`.

Первый запуск устанавливает зависимости и поэтому занимает дольше последующих. Проекты сохраняются только на компьютере проверяющего в `~/.zarya/projects`.

## Запуск для разработки

```bash
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python start_zarya.py
```

Тесты:

```bash
pytest -q
```

## Сборка архива для передачи

```bash
python scripts/build_review_package.py
```

Готовый ZIP появляется в папке `dist`.
