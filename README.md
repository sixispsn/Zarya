# Заря

Инженерный калькулятор внутреннего водоснабжения и канализации по СП 30.13330.2020.

Расчётный модуль для проектирования зданий: водопотребление, полив, водостоки, счётчики, ВПВ, теплоизоляция, насосные установки. Автоматическая генерация пояснительной записки по требованиям ПП-87 (раздел 5, подраздел 2).

## Технологии

- **Backend:** Python 3.12, FastAPI
- **БД:** PostgreSQL 15
- **Frontend:** HTML + JavaScript (временно), переезд на Vue/React позже
- **Документы:** python-docx, pdfplumber

## Запуск (для разработки)

```bash
# Активировать виртуальное окружение
source venv/bin/activate

# Запустить сервер
uvicorn app.main:app --reload
```

Сервер будет доступен по адресу http://localhost:8000.
